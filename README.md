## Features

- Elegant royal Indian heritage aesthetic
- Live countdown timer to launch
- Email signup form with success state
- Animated steam effect on chai cup
- Gold ornamental accents and decorative dividers
- Fully responsive (mobile, tablet, desktop)
- Smooth fade-in animations

## Tech Stack

- React 19
- CSS3 (no external UI libraries)
- Google Fonts (Cormorant Garamond + Inter)

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Deployment

Build for production:

```bash
npm run build
```

Deploy to Vercel, Netlify, or any static hosting.
